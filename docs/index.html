ArbuzTweaker

Твикер для Windows и Dota 2, сделанный для удобной и понятной настройки производительности через конфиги, параметры запуска и видео настройки.

Скачать установщик Скачать zip Все релизы

Что умеет

  • редактирует `autoexec.cfg.txt`
  • настраивает `LaunchOptions` в `localconfig.vdf`
  • редактирует `video.txt`
  • проверяет обновления через GitHub

Установка

  • `ArbuzTweaker-Setup.exe` - установка в `%LocalAppData%\Programs\ArbuzTweaker`
  • `ArbuzTweaker-Portable.zip` - запуск без установки после распаковки

Безопасность

Программа не является читом, не дает игрового преимущества и использует только разрешенные файлы и параметры запуска. Данные пользователя никуда не отправляются и остаются на компьютере.

Про проект

ArbuzTweaker сделан при помощи вайбкодинга и создан для того, чтобы люди могли легко и понятно повышать производительность своих устройств без ручного редактирования десятков файлов.
