# ArbuzTweaker

ArbuzTweaker - Windows и Dota 2 твикер с упором на понятную настройку, безопасные изменения конфигов и удобную публикацию через GitHub Releases.

## Установка

- Установщик: `ArbuzTweaker-Setup.exe`
- Портативная версия: `ArbuzTweaker-Portable.zip`

## Что умеет

- Управление `autoexec.cfg.txt`
- Управление `LaunchOptions` в `localconfig.vdf`
- Управление `video.txt`
- Проверка обновлений через GitHub Releases
- Страница проекта для скачивания последней версии

## Безопасность

- Программа не является читом и не дает игрового преимущества
- Используются только разрешенные файлы и параметры запуска
- Пользовательские данные не отправляются наружу и остаются на компьютере
